    <title>MY First web programming</title>
	<meta charset="utf-8">

- HTML5 文件的基本語法
  - DOCTYPE
  - html
  - head
    - title 
  - body
    - h1 ... h6 
    - p ==>段落
  - 註解與空白字元
  - 註解 <-- -->
